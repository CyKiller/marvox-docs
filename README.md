# Marvox Documentation Site

Documentation website for Marvox Storyworld Production Studio and CharacterOS.

## Features

- Search and quick navigation
- Responsive layout
- Markdown rendering from repo docs

## Getting Started

```bash
cd docs-site
npm install
npm run dev
```

Docs site: `http://localhost:3001`

## Content Model

Docs pages are rendered from markdown in the repo root. Edit the markdown files and the docs site reflects the changes.
